# Student Report System

Student Report System is a Python-based console application for managing student data and grades. The program supports CRUD operations, data validation, ranking, searching, JSON file storage, PDF report export, and undo functionality.

## Features
- Add, update, delete, and view student data
- Process and validate student grades
- Display student rankings
- Search students by name
- Undo last action (Stack)
- Save and load data using JSON
- Export student reports to PDF

## Data Structures & Algorithms
- List
- Dictionary
- Tuple
- Set
- Linked List
- Stack
- Queue
- Recursion
- Merge Sort
- Binary/Recursive Search
